# 📊 Sales Performance Dashboard — Excel Project

An interactive Excel Dashboard built using Pivot Tables and Charts — analyzing 2024-2025 sales data across regions, product categories, and monthly trends with key business KPIs.

## 🗂️ Sheets Structure

| Sheet | Description |
|---|---|
| `Monthly_Sales` | Month-wise sales data |
| `High_value_Customers` | Top customers by purchase value |
| `Pivot_Table_Region_Wise` | Region-wise sales pivot analysis |
| `Charts` | All charts used in dashboard |
| `Dashboard` | Final interactive dashboard |

## 🛠️ Tech Stack

- **Tool:** Microsoft Excel
- **Features:** Pivot Tables, Pivot Charts, Dashboard Design, KPI Cards, Slicers, Conditional Formatting
- **Charts Used:** Bar Chart, Pie Chart, Line Chart
